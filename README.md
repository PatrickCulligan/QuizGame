# Real Kahoot-Style Quiz Clone Starter

Features:
- AI quiz generation
- Host screen
- Game PIN
- Lobby
- Live question flow with SignalR
- Player mobile page
- Final leaderboard

## Run

```bash
dotnet restore
dotnet run
```

Open:
- `/Admin`
- `/Host`
- `/Join`

## Notes

Add your OpenAI API key to `appsettings.json` before using AI generation.
This is a starter structure meant to feel like a Kahoot-style app.
